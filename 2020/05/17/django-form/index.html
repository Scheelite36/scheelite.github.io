表单相关介绍与使用

表单的介绍

常见的表单元素：

  1. 表单标签——<form>
  2. 单行文本框/多行文本框
  3. 选择（单选、多选、下拉选择）
  4. 隐藏表单域
  5. 表单按钮
  6. 文件上传框

<form>标签：

  1. action——表单提交的URL地址
  2. method——表单请求的方式（GET/POST）
  3. enctype——请求内容的形式
    application/x-www-form-urlencoded
   multipart/form-data（用于上上传文件，如word、excel等）

文本标签：

  1. textarea——多行文本

  2. 单行文本（对应input中的type值）

    text password email url number color 时间日期等（date、month、week、time、date、datetime、datetime-local）

选择标签：

  1. 单选多选radio checkbox

  2. 下拉选择

    其他：file上传文件 button hidden隐藏表单域

在视图中获取表单值：

  1. GET请求
    request.GET.get('name', None)
  2. POST请求
    request.POST.get('name', None)
